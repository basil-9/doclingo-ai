# DocLingo AI

AI document tools platform for translating, converting, summarizing, and preparing files.

## Clean V5

- Merge PDF: real browser-based merging with add-one-by-one support.
- Split PDF: real browser-based extraction by pages/ranges.
- PDF tools run in the browser without server upload.

## Netlify

- Build command: `npm run build`
- Publish directory: `dist`
