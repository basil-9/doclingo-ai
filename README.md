# DocLingo AI

AI document tools platform for translating, converting, summarizing, and preparing files.

## Clean V6

- Merge PDF: browser-based merging with add-one-by-one support.
- Split PDF: browser-based extraction by pages/ranges.
- Remove Pages: remove selected pages from a PDF.
- Rotate PDF: rotate all pages or selected pages.
- PDF tools run in the browser without server upload.

## Netlify

- Build command: `npm run build`
- Publish directory: `dist`
