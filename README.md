# DocLingo AI

AI document tools platform for translating, converting, summarizing, and preparing files.

## Local setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Netlify

- Build command: `npm run build`
- Publish directory: `dist`

## Current stage

This is a frontend MVP/prototype. Real backend processing will be added tool by tool.

## Brand direction

- Clean global SaaS style.
- Arabic/English from day one.
- Privacy-first messaging.
- Available / Beta / Coming Soon tool statuses.
