# DocLingo AI V18.1 - AI Controls & Diagnostics

## Added
- AI Status button to verify that the Netlify Function is deployed.
- Maximum AI PDF size: 15 MB.
- Extract Questions options: 5/10/15 questions and easy/medium/hard.

## Required Netlify variables
```text

OPENAI_MODEL=gpt-5-mini
```
