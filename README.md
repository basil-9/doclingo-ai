# DocLingo AI V24 — Supabase Auth

## Added
- Google sign-in through Supabase Auth.
- Email and password registration/sign-in.
- Magic Link sign-in.
- Email confirmation support.
- Auth session restoration and sign-out.
- Account panel with current local credits, usage and history counts.
- Existing Multi-AI, PDF tools, Arabic PDF export and premium UI preserved.

## Required Netlify variables
```text
VITE_SUPABASE_URL
VITE_SUPABASE_ANON_KEY
CLOUDFLARE_API_TOKEN
CLOUDFLARE_ACCOUNT_ID
GEMINI_API_KEY
```

## Supabase
Configure Site URL and Redirect URL, enable Email and Google providers, and use only the publishable key in the browser.
